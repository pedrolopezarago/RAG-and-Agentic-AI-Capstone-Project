# RAG and Agentic AI Capstone Project

This repository contains my work for the **RAG and Agentic AI Capstone Project**, part of Coursera's IBM RAG and Agentic AI specialization.  
The project is organized into five modules, each focusing on a key part of building modern AI systems with generative AI, retrieval, agents, tools, and MCP.

## DISCLAIMER

This is part of Coursera's IBM RAG and Agentic AI Capstone Project by IBM on Coursera. Most scripts within this repository were run on IBM SkillsBuild platform and therefore for their local execution, code may have to be changed.


## Project Goals

The goal of this project is to end up building an AI-powered **Restaurant Recommendation System** that delivers intelligent, personalized suggestions.

## Tools/Software used 

* **Python** - for implementing AI workflows, command-line interfaces, and data processing.
* **LangChain** - for building and orchestrating AI agents.
* **LangGraph** - for visualizing and coordinating multi-agent workflows.
* **Gradio** - for building interactive chatbot interfaces.
* **IBM watsonx.ai** - for multimodal AI and LLM-based workflows. 
* **FAISS** - for constructing multimodal vector indexes.
* **MCP** - Model Context Protocol for integrating agents, tools, and retrieval systems.
* **Jupyter Notebooks** - for experimenting and running Python-based labs.
* **Git/GitHub** - for version control and managing project artifacts 
* **JSON** 


## Modules

### Module 1: Build a Structured Generative AI Application
In this module, raw and unstructured restaurant data are turned into structured formats using an LLM. The goal of this module is to turn unstructured data into structured JSON objects.

### Module 2: Design a Multimodal RAG System
This module implements a Multimodal Retrieval-Augmented Generation (RAG) system to build a similarity retrieval index. The system embeds and indexes both textual data and visual components to enable cross-modal semantic search.
It's possible some of the labs on this module don't work as expected as they were run on IBM Skills Build cloud environment.
